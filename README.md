Credit Card Fraud Detection – DecodeLabs Project 2
Project Overview
This project demonstrates the process of detecting fraudulent credit card transactions through data analysis and machine learning. The dataset was explored, cleaned, and analyzed before building a predictive model to classify genuine and fraudulent transactions.
Project Objectives
Understand the structure of the transaction dataset.
Perform data cleaning and preprocessing.
Check and handle missing values.
Study the distribution of fraudulent and legitimate transactions.
Build a machine learning model for fraud detection.
Assess the model's performance using different evaluation metrics.

Project Workflow
Load the transaction dataset.
Explore the data using Exploratory Data Analysis (EDA).
Clean and preprocess the dataset.
Divide the data into training and testing sets.
Train the machine learning model.
Test the model on unseen data.
Analyze predictions and model performance.
Performance Metrics
The model was assessed using the following metrics:
Accuracy
Precision
Recall
F1-Score
Confusion Matrix
Project Outcome
The developed model was able to distinguish between normal and fraudulent transactions with good performance. The analysis also highlighted important patterns within the transaction data, making the dataset suitable for fraud detection tasks.
